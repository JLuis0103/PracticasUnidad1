Tipos de datos en MySQL

varchar

Se usa para almacenar cadenas de caracteres. Una cadena es una secuancia de caracteres. Se colocca entre comillas (simples): 'Hola'
El tipo "varchar" define una cadena de longitud variable en la cual determinamos el maximo de caracteres. Puede guardar hasta 255 caracteres. Para almacenar cadenas de hasta 30 caracteres, definimos un campo de tipo varchar(30).
